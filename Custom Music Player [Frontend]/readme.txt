To Run:
1. Make sure you have Java and JavaFx installed.
2. Edit the build.bat and run.bat files with the absolute path location of your JavaFx directories.
3. Execute ./build in powershell/cmd from the root directory to compile and run.
4. Execute ./run  in powershell/cmd from the root directory if already compiled.
5. Edit .bat to .sh and make necessary changes for macOS and Unix.
